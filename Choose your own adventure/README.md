# 🗺️ Text-Based Adventure Game (Python)

## 📌 Description
This is a simple **text-based adventure game** built using Python.  
The player makes decisions at different stages of the story, and each choice leads to a different outcome — win or lose.

The game demonstrates how decision-making works in programming using conditional statements.

---

## 🎯 Features
- Interactive story-based gameplay 📖
- Multiple decision paths
- Win and lose scenarios
- User input handling
- Beginner-friendly logic using `if-else`

---

## 🕹️ How to Play
1. Run the program.
2. Enter your name.
3. Follow the story and make choices:
   - Choose directions (left/right)
   - Decide actions (walk/swim, cross/back, yes/no)
4. Each decision affects the outcome.
5. Try different paths to win the game!

---

## 🧠 Game Logic
- Uses nested `if-else` conditions to control story flow
- User inputs determine the next scenario
- Invalid inputs lead to losing the game
- Multiple endings:
  - 🏆 Win (get gold)
  - ❌ Lose (various scenarios)

---

**Easy 🟢**

Suitable for:

- Practice for conditional statements
- Understanding decision trees

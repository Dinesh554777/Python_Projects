# 🧠 Python Quiz Game

## 📌 Description
This is a simple command-line based Quiz Game built using Python.  
The program asks the user a series of basic computer-related questions and calculates the final score based on correct answers.

It is designed for beginners to understand:
- User input handling
- Conditional statements
- Basic scoring logic
- String manipulation in Python

---

## 🎯 Features
- Interactive quiz experience
- Case-insensitive answer checking
- Score tracking system
- Percentage calculation at the end
- Beginner-friendly code structure

---

## 🕹️ How to Play
1. Run the Python file.
2. Enter **yes** to start the quiz.
3. Answer the questions one by one.
4. Get your final score and percentage at the end.
